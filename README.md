GameBalance

Веб-приложение для отслеживания игрового времени и анализа его влияния на эмоциональное состояние подростков. Позволяет фиксировать сессии, вести статистику и использовать таймер Pomodoro для самоконтроля.

Описание работы

Пользователь регистрируется и входит в аккаунт(по желанию).
На странице ввода данных указывает название игры, время в часах и своё настроение после сессии.
Информация сохраняется в базе данных. На странице статистики отображаются личные и анонимные данные.
Встроенный таймер помогает контролировать время в игре.

Необходимые библиотеки

fastapi
uvicorn
sqlalchemy
psycopg2-binary
python-multipart
jinja2

Установка

pip install fastapi uvicorn sqlalchemy psycopg2-binary python-multipart jinja2

Запуск веб-сайта

Запустите базу данных PostgreSQL. Убедитесь, что настройки подключения в файле database.py верны.
Запустите инициализацию базы данных: python init_db.py.
Запустите веб-приложение: uvicorn app:app --reload.
Откройте браузер и перейдите по адресу http://127.0.0.1:8000.
